# WebVentas3d
Proyecto que simula la página web de una empresa (Prosoft Printer), qué se dedica a la venta de productos impresos en 3D.


Comandos para instalar las dependencias:

 npm i express -S   
 npm i ejs -S   
 npm i nodemon -D   
 npm install cookie-session -S   
 npm install ejs -S
